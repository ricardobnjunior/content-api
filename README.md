# FastAPI SQLAlchemy Project

A RESTful API built with **FastAPI** and **SQLAlchemy**, following modern Python best practices. This project provides a structured backend with authentication, data persistence, and a clean API interface.

---

## Technologies Used

| Technology | Version | Purpose |
|------------|---------|---------|
| Python | 3.10+ | Runtime |
| FastAPI | 0.100+ | Web framework |
| SQLAlchemy | 2.0+ | ORM / database layer |
| Uvicorn | 0.22+ | ASGI server |
| Alembic | 1.11+ | Database migrations |
| Pydantic | 2.0+ | Data validation |
| pytest | 7.0+ | Test framework |
| SQLite / PostgreSQL | -- | Database backend |

---
